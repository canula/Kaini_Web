Getting Started

TC-01 Tone Calculator is an algorithmic melody generator available as a free web app, Android app, and desktop application. It turns your ideas into melodies — no music theory knowledge required.

Novice Music Makers

Explore the fundamental elements that make a melody work:

  • - How do different scales create different moods?
  • - What makes a melody move (intervals, contour, rhythm)?
  • - How do ornaments add expression?
  • - What creates musical structure and cohesion?

TC-01 lets you experiment with each element independently, building your musical intuition through hands-on exploration.

Experienced Music Makers

Quickly explore new musical possibilities and break out of patterns:

  • - Generate fresh melodic ideas instantly
  • - Explore unfamiliar scales, modes, and rhythmic patterns
  • - Discover unexpected combinations
  • - Overcome creative blocks with algorithmic inspiration
  • - Rapidly prototype musical concepts

Two Ways to Create

Phrase Mode generates complete 8-bar compositions from professional templates. Free Mode lets you build bar-by-bar with full creative control. Both modes are covered in detail below.

Phrase Mode

Phrase Mode generates complete 8-bar musical compositions based on professional compositional templates. Perfect for quick inspiration or learning phrase structure.

Create an 8-Bar Composition in 4 Clicks

  1. 1.
    Enable Phrase Mode — Toggle in the transport bar (dual-purpose button: shows "Phrase" when Select is off). Activates the Phrase Mode generation system.
  2. 2.
    Click TR-8 (Phrase Transform) — Opens the Phrase Mode modal with 100+ presets.
  3. 3.
    Select a Preset — Choose from genres: Classical, Jazz, World, Electronic, Experimental. Examples: "Baroque Dance", "Bebop Lines", "Gamelan", "Ambient Drift".
  4. 4.
    Press Play — Listen to your 8-bar composition.
Refining Your Phrase (Advanced Sliders)

Advanced Sliders (in TR-8 modal):

  • - Motivic Return: How often Bar 1's idea comes back
  • - Development Intensity: How much variation in middle sections
  • - Energy Direction: Build up or wind down
  • - Resolution Strength: How definitively it ends
  • - Harmonic Motion: How much key/scale changes

Or Edit in Main UI:

  • - Bars 1 & 2 are editable
  • - Changes affect how the phrase develops across all 8 bars
Tip: Phrase Mode presets override Bars 1-2, so start with a preset FIRST, then customize Bar 2 for variation.

Free Mode

Free Mode gives you full manual control to build melodies bar-by-bar. Perfect for detailed composition and experimentation.

Build a Melody from Scratch

  1. 1.
    Start with Bar 1 — Bar 1 is active by default (click to select if needed).
  2. 2.
    Create Your Foundation Settings:
    • - TR1 (Key): Choose tonal center (e.g., C)
    • - TR2 (Scale): Pick a scale (e.g., Major)
    • - TR3 (Motion): Set interval style (e.g., Stepwise)
    • - TR4 (Contour): Choose melodic shape (e.g., Arch)
    • - TR5 (Rhythm): Select rhythm pattern
    • - TR7 (Instrument): Pick sound + audio settings
  3. 3.
    Copy Bar 1 to Bar 2 — Drag Bar 1 onto Bar 2 to copy all transform settings (or manually configure Bar 2).
  4. 4.
    Tweak a Setting in Bar 2 — Change Motion from "Stepwise" to "Skip", or Contour from "Arch" to "Valley". This creates variation while maintaining coherence.
  5. 5.
    Press Play — Listen to your 2-bar melody, then continue building.
Bar Building Techniques

Call-and-Response (2 Bars)

  • - Bar 1: "Question" — Piano, Rising contour, Stepwise motion
  • - Bar 2: "Answer" — Piano, Falling contour, Skip motion
Music Theory: Call-and-response creates tension and release. The "call" poses a musical question, the "response" provides an answer.

Motivic Development (4 Bars)

  • - Bar 1: Original idea (C Major, Arch contour, Stepwise)
  • - Bar 2: Same key/scale, different rhythm
  • - Bar 3: Same rhythm as Bar 2, different contour
  • - Bar 4: Return to Bar 1's rhythm, new key (G Major)
Music Theory: Repetition creates familiarity, variation creates interest. This balance is the essence of memorable melodies.

Contour Conversations

  • - Bar 1: Rising contour — builds energy
  • - Bar 2: Falling contour — releases energy
  • - Bar 3: Arch contour — rises and falls (self-contained)
  • - Bar 4: Plateau — provides stability/rest
Music Theory: Rising = anticipation, Falling = resolution, Arch = complete gesture, Plateau = contemplation.

The Bar System

TC-01 operates on an 8-bar structure per section. Each bar can be in one of four states:

Active

Plays back

Inactive

Stored, muted

Selected

Being edited

Playing

Live playback

Working with Bars

  • Tap/Click bar — Select and view its transforms
  • Drag bar to another bar — Copy all transforms to target bar
  • Drag bar outside the grid — Bin icon appears with red flash; release to delete

Playback Rules

  • - If no bars are marked (shifted): Plays from Bar 1 to last active bar
  • - If bars are marked (orange border): Plays only marked bars
  • - Each bar has independent transform settings

The Section Builder

TC-01's Section Builder lets you organize 8-bar segments into larger structures — think of sections as building blocks for complete songs.

Each section contains:

  • - 8 independent bars (each with its own transform settings)
  • - A unique name (e.g., "Verse", "Chorus", "Bridge")
  • - Independent playback and editing
Building a Song Structure (Example)
  • - Section 1: Intro (4 bars active)
  • - Section 2: Verse (8 bars active)
  • - Section 3: Chorus (8 bars active)
  • - Section 4: Bridge (8 bars active)
  • - Section 5: Outro (4 bars active)
Creating, Renaming & Deleting Sections

Creating:

  1. 1. Click/Tap the "+" button
  2. 2. New section appears with default name "Section [number]"
  3. 3. Drag section to another section to copy transforms
  4. 4. All 8 bars start inactive — click/tap bars to activate them

Renaming:

  1. 1. Click/Tap on the section
  2. 2. Go to Hamburger Menu > Rename Current Section
  3. 3. Enter new name (e.g., "Verse 1", "Chorus A", "Solo")

Deleting:

Drag section chip outside the carousel area — Bin icon appears with red pulse animation — release to delete.

Warning: Cannot delete the last active section. Deleted sections cannot be recovered.

Navigating & Playing Across Sections

Section Carousel:

  • - Shows all sections as chips/cards
  • - Currently selected section highlighted with orange border
  • - During playback, carousel auto-scrolls to currently playing section (green glow)
  • - Navigate with chevron buttons (< >) or swipe/scroll

Playback Across Sections:

  • - TC-01 plays sections in sequence
  • - Playback flows through them automatically
  • - Green glow indicates currently playing section + bar
Tips: Name sections clearly ("Intro", "Verse 1", "Pre-Chorus"). Save often with Ctrl+S. Test individual sections before combining. Build gradually — start with 1-2 sections, get them right, then add more.

Select Mode

Select Mode lets you mark specific bars or sections for focused playback. Enable it via the "Select" button in the transport controls.

When Select Mode is ON:

  • - Tap/Click any active bar or section to toggle its "marked" state (orange border)
  • - The dual-purpose button switches to Loop mode (for continuous repeat playback)
  • - Only marked bars/sections will play

When Select Mode is OFF:

  • - Normal tap behavior (select to view and edit transforms)
  • - The dual-purpose button shows Phrase Mode toggle

Important: Select Mode is disabled (grayed out) when Phrase Mode is active. Enabling Phrase Mode automatically clears all marked bars and disables Select Mode.

Transform Reference: TR1-TR2 (Key & Scale)

TR1: Key Transform

Sets the tonal center — the "home note" around which your melody revolves.

Options: C, C♯/D♭, D, D♯/E♭, E, F, F♯/G♭, G, G♯/A♭, A, A♯/B♭, B

Music Theory: The key doesn't just determine pitch — it affects the "color" and "feel" of the melody. Different keys suit different instruments and vocal ranges.

TR2: Scale Transform

Defines which notes are available within the chosen key.

Common Scales

Major: Bright, happy, stable (do-re-mi-fa-sol-la-ti-do)

Natural Minor: Dark, melancholic, introspective

Harmonic Minor: Exotic, dramatic (raised 7th creates tension)

Melodic Minor: Smooth ascending, natural descending

Modes

Dorian: Minor with raised 6th (jazz, folk)

Phrygian: Minor with lowered 2nd (Spanish, flamenco)

Lydian: Major with raised 4th (dreamy, floating)

Mixolydian: Major with lowered 7th (rock, blues, folk)

Aeolian: Natural minor (sad, contemplative)

Locrian: Diminished, unstable (rarely used)

Music Theory: Scales constrain your melody to specific intervals, creating coherence. Major = consonance. Minor modes = more dissonance and tension.

Transform Reference: TR3-TR4 (Motion & Contour)

TR3: Motion Transform

Controls how notes move — the interval relationships between consecutive notes.

Stepwise (50%+)

Small intervals — 2nds, 3rds. Smooth, singable, easy to follow.

Skip (50%+)

Medium leaps — 4ths, 5ths, 6ths. Energetic, interesting, confident.

Leap (50%+)

Large jumps — 7ths, octaves. Dramatic, attention-grabbing, virtuosic.

Mixed

Combination of all types. Balanced, unpredictable, natural speech-like.

Music Theory: Stepwise motion = vocal melodies, lyrical lines. Leaps = instrumental showpieces. Classical rule: After a leap, resolve by step in opposite direction.

TR4: Contour Behavior

Shapes the overall melodic direction and arc.

Rising

Generally ascending — builds tension, energy, anticipation.

Falling

Generally descending — releases tension, calming, resolving.

Arch

Rise then fall — complete gesture, self-contained phrase.

Valley

Fall then rise — sigh, question-answer within single bar.

Plateau

Mostly static, small variations — meditative, hypnotic, minimalist.

Advanced Contour Features

Gravity Points: Set specific pitch targets that the melody gravitates toward — creates intentional melodic destinations and focal points.

Multi-Bar Contours: Extend contour shapes across up to 4 bars — creates larger-scale melodic arcs that span multiple bars.

Music Theory: Contour creates the emotional shape of a phrase. Combine with motion: Arch + Stepwise = gentle/lyrical, Rising + Leap = urgent/climactic.

Transform Reference: TR5-TR6 (Rhythm & Ornaments)

TR5: Rhythm System

Assigns duration patterns and timing to notes.

  • - Preset Rhythms: Pre-defined patterns (quarter notes, eighth notes, syncopation, etc.)
  • - Custom Patterns: Create your own sequences
  • - Bar Length (Beats Per Bar): Choose 2, 3, or 4 beats per bar
    • 2/4 time = March feel (strong-weak)
    • 3/4 time = Waltz feel (strong-weak-weak)
    • 4/4 time = Standard rock/pop (strong-weak-medium-weak)
  • - Anacrusis (Pickup Notes): Start melody before the first downbeat — creates anticipation and forward momentum
  • - Preview: Hear rhythm with metronome before applying
Music Theory: Rhythm is HALF the melody. Same pitches with different rhythms = completely different songs. Syncopation = energy, surprise.

TR6: Ornamentation

Adds decorative embellishments to individual notes (Baroque-era technique).

  • - Trill: Rapid alternation with upper neighbor note
  • - Mordent: Quick lower neighbor + return
  • - Turn: Four-note figure circling the main note
  • - Appoggiatura: "Leaning note" that resolves down
  • - Grace Note: Quick preceding ornamental note
Music Theory: Ornaments add expression without changing the underlying melody. Use sparingly on important notes. Overuse = cluttered.

Transform Reference: TR7 (Instrument)

TR7: Instrument Transform

Selects the sound source and configures audio parameters for the selected bar.

  • - Instrument: Piano, Bass, and more (plus custom instruments with Pro license)
  • - Octave Shift: Transpose the bar up or down by octaves
  • - Volume: Loudness (0-100%)
  • - Reverb: Spatial depth, room size (0-100%)
  • - Stereo Pan: Position in stereo field (L = left, C = center, R = right)
Custom Instruments (Pro)

With a Pro license, you can add your own audio samples as custom instruments. Load any WAV or MP3 file and TC-01 will map it across the pitch range for use in your melodies.

Music Theory: Timbre (instrument choice) affects mood and genre association. Piano = versatile, percussive attack. Bass = low register, foundational. Different bars can use different instruments for textural variety.

Transform Reference: TR8 (Phrase Mode)

TR8: Phrase Transform

Active only in Phrase Mode. Generates complete 8-bar compositions based on phrase structure templates.

Features:

  • - 100+ Presets: Classical, Jazz, Popular, World, Electronic, Experimental
  • - Form Templates: ABA, AABA, Rondo, and more — mirrors how classical composers built larger forms from small motifs
  • - Editable Bar 2: Customize the "response" to Bar 1's "call"
Advanced Parameters

Motivic Return: How often Bar 1's idea recurs throughout the 8-bar phrase.

Development Intensity: How much variation is applied to middle sections.

Energy Direction: Crescendo (build up) or Decrescendo (wind down).

Resolution Strength: How strongly and definitively the phrase ends.

Harmonic Motion: Amount of key/scale modulation across the phrase.

DNA Presets by Genre

Classical: Baroque Dance, Romantic Nocturne, Minimalist Pulse...

Jazz: Bebop Lines, Cool Jazz, Modal Exploration...

World: Gamelan, Raga-inspired, Celtic Air...

Electronic: Ambient Drift, Techno Sequence, Glitch Melody...

Experimental: Atonal, Aleatoric, Micropolyphony...

Music Theory: Phrase Mode uses formal structures (ABA, AABA, Rondo, etc.). This mirrors how classical composers built larger forms from small motifs — providing structural coherence across all 8 bars.

MIDI Import

Bring in MIDI files from your DAW or other apps. Edit, transform, and build on melodies you've already started.

How It Works

  1. 1.
    Load a MIDI File — Click "Load from file" and select a .mid file from your device.
  2. 2.
    Track Selection — Single-track files import automatically. Multi-track files let you choose which track to use.
  3. 3.
    Fragment Analysis — TC-01 analyzes the MIDI data and breaks it into fragments on a pad grid.
  4. 4.
    Property Extraction — Musical properties (key, scale, motion, contour, rhythm) are extracted and can be applied to bars.
Managing MIDI Files
  • - Imported files are listed with filename, upload timestamp, and track count badge
  • - Delete individual MIDI files with confirmation
  • - Browse fragments after at least 1 file is uploaded
  • - All files are associated with your current project

Voice Recording

Hum a melody or sing a tune and TC-01 transcribes it into notes instantly. You can also import an audio file instead of recording live.

How It Works

  1. 1.
    Choose a Voice Preset — Select your vocal range (Soprano, Alto, Tenor, Bass, etc.) to help constrain pitch detection.
  2. 2.
    Record — Optional metronome count-in using your project's BPM. Real-time waveform visualization while recording.
  3. 3.
    Pitch Detection — TC-01 auto-converts your recording to MIDI notes with per-note confidence visualization.
  4. 4.
    Quantization — Detected notes are quantized to the grid and become usable melody fragments.
Detection Confidence

Each detected note shows a confidence indicator:

  • - Green (95%+) — High confidence detection
  • - Amber (90-95%) — Good detection
  • - Red (<90%) — Lower confidence, may need editing

Microphone audio is processed entirely on your device and never transmitted to any server. Recordings are stored temporarily with a 24-hour expiry.

Note Edit Mode

A tracker-style note editor that lets you modify individual notes within a bar. Open it by double-clicking/double-tapping any active bar.

Features

  • Two Edit Modes:
    • - Chromatic: Edit notes using all 12 semitones
    • - Scale-Relative: Edit notes constrained to the current bar's scale
  • Undo/Redo: Per-bar edit history (up to 16 steps). History resets when switching bars.
  • Preview: Audition your edited notes in real time.
  • Microtonal Support: Available when experimental microtonal mode is enabled.
Pro Feature: Note Edit Mode requires a Pro license. Close the editor with ESC or by clicking outside the modal.

Exporting

TC-01 supports three export formats. Access export options from the Hamburger Menu.

MIDI (.mid)

Standard MIDI file for use in DAWs (Ableton, Logic, FL Studio, etc.). Contains note data, timing, and velocity. Use for further editing, arrangement, and mixing.

Audio (.wav)

High-quality WAV audio file. Direct recording of synth output with all effects applied. Use for sharing, sampling, or archiving.

Preset (.kai)

TC-01 project file (custom format). Contains all transform settings, bar configurations, and sections. Use for saving projects, sharing templates, or building a preset library.

Export Scope Options:

  • - Current Playback: Respects Loop/Phrase Mode/Play Range
  • - All Active Bars: Exports everything marked as active (ignores playback state)

Pro Feature: MIDI and WAV export require a Pro license. Preset (.kai) save/load also requires Pro.

For Beginners

  • Start Simple

    One bar, Major scale, Stepwise motion, Quarter-note rhythm. Add complexity gradually.

  • Use the Metronome

    Helps you understand rhythm patterns, especially syncopation.

  • Preview Rhythms

    TR5 has a preview button — always listen before applying.

  • Experiment with Modes

    Don't stay in Major/Minor. Try Dorian or Mixolydian for fresh colors.

  • Save Often

    Ctrl+S saves your preset as a .kai file. Save versions as you experiment.

For Advanced Users

  • Random Exploration

    Pick random settings across all transforms — discover happy accidents.

  • Polymeter Experiments

    Use different bar lengths per bar (TR5). E.g., Bar 1 = 4/4, Bar 2 = 3/4 for complex polyrhythmic textures.

  • Strategic Ornamentation

    Don't ornament every note — target key moments (cadences, long notes).

  • Phrase Mode for Development

    Use TR8 to expand 2-bar ideas into full 8-bar compositions.

  • Contour + Motion Combos

    Arch + Skip = dramatic arc. Plateau + Leap = restless energy. Valley + Stepwise = gentle sigh.
