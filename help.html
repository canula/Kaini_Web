Getting Started

TC-01 is an algorithmic melody generator. You configure musical parameters — key, scale, motion, contour, rhythm, ornamentation, and instrument — using eight transforms (TR1–TR8), and TC-01 generates melodies from those settings. Each of the 8 bars in a section holds its own independent transform configuration.

In Free Mode (default) you control every bar manually. In Phrase Mode you set up Bar 1 and Bar 2 as seed material and TC-01 algorithmically generates Bars 3–8 using a form template and five development sliders (TR8).

Available as a free cross-platform web app (PWA), Android app, and Windows desktop application. Features marked Pro require a license.

The interface has four rows. The Section Carousel at the top lets you switch between sections. The Bar Grid shows 8 bars — tap to select, drag between bars to copy settings, or drag outside the grid to delete a bar. Sections work the same way in the carousel.

Transport Bar

  • - Menu: Access settings, rename sections, export, save/load sessions, and activate Pro.
  • - Select: Enter Select Mode to mark specific bars for focused playback. Tap bars to toggle them on or off — only selected bars will play.
  • - Play/Stop: Start or stop playback.
  • - Phrase/Loop: Toggle between Phrase Mode (algorithmic generation) and Loop Mode (continuous looping of active bars).
  • - Met: Toggle the metronome on or off during playback.

Colour coding: green = playing, orange = selected, full opacity = active, greyed out = inactive.

Phrase Mode

Tap the Phrase/Loop button in the transport bar to enable Phrase Mode, open TR8, and pick a preset. This populates Bars 1–2 as seed material and sets five development sliders that control how Bars 3–8 are algorithmically generated. You can override any setting — edit Bar 1/2 transforms, adjust sliders — then press Play.

Development Sliders

  • - Motivic Return: How often Bar 1's motif recurs (low = new material, high = callbacks)
  • - Development Intensity: Variation in middle bars (low = subtle, high = significant)
  • - Energy Direction: Build up or wind down toward Bar 8
  • - Resolution Strength: How definitively the phrase ends
  • - Harmonic Motion: Key modulation across bars (low = stays in key, high = modulates)

Form Templates

Each preset uses a form template — ABA, AABA, Rondo, or Through-composed — that assigns structural roles (statement, contrast, return, etc.) to each bar pair. 100+ presets across Classical, Jazz, World, Electronic, and Experimental categories.

Free Mode

The default mode. Configure each bar independently using TR1–TR7. A typical workflow: set up Bar 1, drag it to Bar 2 to copy, then tweak a parameter or two for variation. Repeat to build out your composition. Use Loop Mode (transport bar) to continuously loop playback through your active bars.

Bars & Sections

Each section holds 8 bars with independent settings. Playback runs left to right through active bars, then advances to the next section. Tap "+" in the section carousel to add new sections; rename or delete sections via Menu.

Select Mode

Tap Select in the transport bar to enter Select Mode. Tap individual bars to mark or unmark them — only marked bars will play during playback. This is useful for auditioning specific bars or isolating a passage. Tap Select again to exit and return to normal playback.

Transforms

TR1: Key

Root note (C through B). Changing key transposes the output.

TR2: Scale

Constrains generated notes to the selected scale. Includes Major, Minor, Harmonic Minor, Melodic Minor, and all modes (Dorian, Phrygian, Lydian, Mixolydian, Locrian). Microtonal scales are also available — see Microtonal Support.

TR3: Motion

Interval size — Stepwise (2nds/3rds), Skip (4ths/5ths/6ths), Leap (7ths+), or Mixed.

TR4: Contour

Pitch direction — Rising, Falling, Arch, Valley, or Plateau.

Gravity Points: Set specific pitch targets within a bar that the melody is drawn toward, creating more nuanced contour shapes.

Multi-Bar Contours: Span a single contour shape across up to 4 consecutive bars for longer, more cohesive melodic arcs.

TR5: Rhythm

Choose a preset pattern or build your own on a subdivision grid. Set bar length (2/4, 3/4, 4/4) — bars can differ for polymetre. Preview a rhythm before applying it.

Anacrusis: Add pickup notes before the downbeat of a bar. Enable this in TR5 to create phrases that lead into the next bar.

TR6: Ornaments

Embellishments applied to notes — Trill, Mordent, Turn, Appoggiatura, or Grace Note.

TR7: Instrument

Sound source, octave shift, volume, reverb, and stereo pan. Choose from built-in instruments (Piano, Bass, etc.).

Custom Samples Pro — Load your own WAV or MP3 files as instruments. Open TR7 and use the import option to load a sample from your device. The sample will be mapped across the pitch range automatically.

TR8: Phrase

Only active in Phrase Mode. Controls the development sliders and presets. See Phrase Mode above.

Microtonal Support

TC-01 supports microtonal scales beyond standard 12-TET (12-tone equal temperament) tuning. This lets you explore tuning systems used in non-Western music traditions, experimental composition, and alternative temperaments.

Access microtonal scales through TR2: Scale. When a microtonal scale is selected, the pitch output adjusts accordingly — all other transforms (motion, contour, rhythm, ornaments) continue to work as normal with the microtonal pitch set.

MIDI Import

Load a .mid file and TC-01 segments it into bar-length fragments, extracting key, scale, motion, contour, and rhythm as transform settings you can apply to any bar. Use this to build on melodies from your DAW or other apps.

Voice Recording

Hum, sing, or whistle a melody — or import an audio file — and TC-01 transcribes it into note data via pitch detection. Select a vocal range preset first for better accuracy. Detected pitches are quantised to the beat grid.

Audio is processed on-device and never transmitted. Recordings expire after 24 hours.

Note Edit Mode Pro

Double-tap any active bar to open a tracker-style note editor. Edit individual notes in Chromatic mode (all 12 semitones) or Scale-Relative mode (only notes from the selected scale). Includes undo/redo and real-time preview as you edit.

Saving & Exporting Pro

Save & Recall Sessions

Save your current session as a .kai project file via Menu or Ctrl+S. This preserves all sections, bars, and transform settings. Load a saved session with Ctrl+O or via Menu.

Export

Via the Menu — export as MIDI (.mid), Audio (.wav), or Preset (.kai project file). Choose to export the current playback range or all active bars.
