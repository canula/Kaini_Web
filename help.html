Getting Started

TC-01 Tone Calculator is an algorithmic melody generator available as a free web app, Android app, and desktop application. It combines eight musical parameters — key, scale, motion, contour, rhythm, ornamentation, instrument, and phrase structure — each controlled through a transform (TR1–TR8).

Core Capabilities

  • - Generate melodies in any key, scale, or mode
  • - Build 8-bar phrases from form templates (ABA, AABA, Rondo, etc.)
  • - Import MIDI files or record vocals for pitch detection
  • - Edit individual notes in a tracker-style editor Pro
  • - Export to MIDI, WAV, or .kai preset format Pro
  • - Organise compositions into named sections

Two Composition Modes

Phrase Mode generates Bars 3–8 automatically from two bars of seed material using form templates and development parameters. Free Mode (default) gives you direct control over every transform in every bar.

Phrase Mode

Phrase Mode generates 8-bar compositions from two bars of seed material. Bar 1 and Bar 2 are your input — a form template and five development parameters handle the rest.

How It Works

  1. 1.
    Enable Phrase Mode — Toggle in the transport bar. This activates TR8 and locks Bars 3–8 to algorithmic generation.
  2. 2.
    Open TR8 — Browse 100+ presets and five development sliders.
  3. 3.
    Select a Preset — Populates Bar 1, Bar 2, and all sliders. Override anything before generating.
  4. 4.
    Press Play — Bars 3–8 are generated and the full phrase plays.

Development Parameters

Five sliders in TR8 control how seed material is developed across Bars 3–8:

Motivic Return

How often Bar 1's motif recurs. Low = more new material; high = frequent callbacks to the opening idea.

Development Intensity

How much variation is applied to middle bars. Low = subtle shifts; high = significant rhythmic and intervallic transformation.

Energy Direction

Dynamic arc of the phrase. "Build" = increasing density and register toward Bar 8. "Wind Down" = the inverse.

Resolution Strength

How definitively the phrase ends. Low = open, unresolved; high = strong cadential resolution to the tonic.

Harmonic Motion

Key/scale modulation across bars. Low = stays in one key; high = modulates to related keys before returning.

Bar 1 and Bar 2 remain editable while Phrase Mode is active — changes are reflected in the generated output.

Free Mode

Free Mode (the default) gives you direct control over all transforms in every bar. Each bar is configured independently; TR8 is disabled.

Workflow

  1. 1.
    Configure Bar 1 — Set key, scale, motion, contour, rhythm, and instrument using TR1–TR7.
  2. 2.
    Duplicate & vary — Drag Bar 1 onto Bar 2 to copy settings, then tweak a parameter or two for variation.
  3. 3.
    Extend — Continue adding bars. Each can share settings with previous bars or diverge entirely.
Composition Patterns

Call-and-Response (2 Bars)

  • - Bar 1: Rising contour, Stepwise motion
  • - Bar 2: Falling contour, Skip motion

Motivic Development (4 Bars)

  • - Bar 1: Original motif
  • - Bar 2: Same key/scale, different rhythm
  • - Bar 3: Same rhythm as Bar 2, different contour
  • - Bar 4: Return to Bar 1's rhythm in a new key

Contour Sequencing (4 Bars)

  • - Bar 1: Rising
  • - Bar 2: Falling
  • - Bar 3: Arch
  • - Bar 4: Plateau

The Bar System

Each section contains 8 bars with independent transform settings (TR1–TR7). Bars have four states:

Active

Included in playback

Inactive

Stored but muted

Selected

Being edited

Playing

Currently sounding

Bar Operations

  • Tap a bar — Select it for editing.
  • Drag onto another bar — Copy all transform settings to the target.
  • Drag outside the grid — Delete the bar (resets to inactive).

Playback runs sequentially from Bar 1 through the last active bar. In Select Mode, only marked bars play.

The Section Builder

Sections are named 8-bar containers (e.g., "Verse", "Chorus") that play sequentially in carousel order. Each holds its own bar configurations.

  • Create — Tap "+" in the carousel. Drag one section onto another to copy all bar settings.
  • Rename — Menu > Rename Current Section.
  • Delete — Drag the section chip outside the carousel. The last section cannot be deleted.

Playback advances automatically from the last active bar of one section to Bar 1 of the next.

Select Mode

Mark specific bars or sections for focused playback. Toggle via the "Select" button in the transport bar.

  • ON: Tap bars/sections to mark them (orange border). Only marked items play. The dual-purpose button switches to Loop mode.
  • OFF: Normal editing behaviour. The dual-purpose button shows the Phrase Mode toggle.

Select Mode is unavailable when Phrase Mode is active.

TR1–TR2: Key & Scale

TR1: Key

Sets the tonal centre (root note). Changing key transposes the output while preserving intervals.

Options: All 12 chromatic pitches (C through B).

TR2: Scale

Defines the available pitch set. Generated notes are constrained to the selected scale.

Scales & Modes

Major (Ionian): W-W-H-W-W-W-H

Natural Minor (Aeolian): W-H-W-W-H-W-W

Harmonic Minor: Natural minor with raised 7th

Melodic Minor: Raised 6th and 7th ascending; natural minor descending

Dorian: Minor with raised 6th

Phrygian: Minor with lowered 2nd

Lydian: Major with raised 4th

Mixolydian: Major with lowered 7th

Locrian: Diminished — lowered 2nd and 5th

TR3–TR4: Motion & Contour

TR3: Motion

Controls interval size between consecutive notes.

Stepwise

Favours 2nds and 3rds — smooth, conjunct lines.

Skip

Favours 4ths, 5ths, and 6ths — wider movement.

Leap

Favours 7ths, octaves, and beyond — angular, disjunct lines.

Mixed

No bias — full range of intervals.

TR4: Contour

Sets the overall pitch direction. TR3 controls interval size; TR4 controls direction.

Rising

Net ascending trajectory.

Falling

Net descending trajectory.

Arch

Ascends then descends.

Valley

Descends then ascends.

Plateau

Minimal pitch movement — rhythm provides the interest.

Advanced Contour Features

Gravity Points: Set specific pitch targets at positions within the bar.

Multi-Bar Contours: Stretch a single contour shape across up to 4 consecutive bars.

TR5–TR6: Rhythm & Ornaments

TR5: Rhythm

Sets the duration pattern for generated notes.

  • - Preset Rhythms: Straight, syncopated, and compound (dotted/triplet) patterns.
  • - Custom Pattern Builder: Toggle beat subdivisions on a grid (up to 4 per beat).
  • - Bar Length: 2/4, 3/4, or 4/4 time. Each bar can differ, enabling polymetre.
  • - Anacrusis: Pickup notes before the first downbeat.
  • - Preview: Audition the pattern with a metronome click before applying.

TR6: Ornaments

Adds melodic embellishments to notes. Each ornament subdivides the note's duration.

  • - Trill: Rapid alternation with the upper neighbour.
  • - Mordent: Main note → lower neighbour → main note.
  • - Turn: Four-note figure circling the main note (upper → main → lower → main).
  • - Appoggiatura: Accented note on the beat resolving stepwise down.
  • - Grace Note: Quick unaccented note before the beat.

TR7: Instrument

Selects the sound and audio settings per bar. Each bar can use a different instrument.

  • - Instrument: Built-in voices (Piano, Bass, etc.) or custom samples Pro
  • - Octave Shift: Transpose up or down by whole octaves.
  • - Volume: Per-bar level (0–100%).
  • - Reverb: Wet/dry mix (0–100%).
  • - Stereo Pan: Left, Centre, or Right.

Custom instruments (Pro): Load any WAV or MP3 — TC-01 pitch-shifts it across the note range.

TR8: Phrase Transform

Active only in Phrase Mode. TR8 assigns each bar a structural role via a form template, then uses five development sliders to vary the seed material. See Phrase Mode for the full workflow and slider details.

Form Templates

  • - ABA: Statement → Contrast → Return → Coda
  • - AABA: Statement → Repeat → Bridge → Return
  • - Rondo: Theme → Episode → Theme → Episode/Coda
  • - Through-composed: Continuous development, no literal repeats.

Presets

100+ presets, each encoding a form template, slider defaults, and recommended TR1–TR6 settings. Select one to populate everything, then override as needed.

Preset Categories

Classical: Baroque, Classical-era, Romantic, and Minimalist forms.

Jazz: Bebop, Cool Jazz, and Modal idioms.

World: Pentatonic, raga-based, and gamelan scales.

Electronic: Ambient, Sequence, and Glitch patterns.

Experimental: Atonal, Aleatoric, and Micropolyphony.

MIDI Import

Import .mid files to extract melodic fragments as transform settings.

  1. 1.
    Load — Select a .mid file. Multi-track files present a track selector.
  2. 2.
    Analyse — TC-01 segments the data into bar-length fragments on a pad grid.
  3. 3.
    Apply — Each fragment's detected key (TR1), scale (TR2), motion (TR3), contour (TR4), and rhythm (TR5) can be applied directly to any bar.

Voice Recording

Hum, sing, or whistle a melody and TC-01 converts it to note data via pitch detection.

  1. 1.
    Select vocal range — Soprano, Alto, Tenor, Bass, etc. Improves detection accuracy.
  2. 2.
    Record — Optional metronome count-in. You can also import an audio file.
  3. 3.
    Review — Detected pitches are quantised to the beat grid. Confidence indicators show detection certainty (green = high, amber = moderate, red = review recommended).

Audio is processed on-device and never transmitted externally. Recordings expire after 24 hours.

Note Edit Mode

Tracker-style note editor — double-tap any active bar to open. Edit in Chromatic (all 12 semitones) or Scale-Relative mode. Includes undo/redo (16 steps), real-time preview, and optional microtonal support.

Pro Feature: Note Edit Mode requires a Pro license.

Exporting

Three formats, accessible from the Menu:

MIDI (.mid)

Standard MIDI for DAW use — note data, timing, and velocity.

Audio (.wav)

WAV recording of the synth output with effects applied.

Preset (.kai)

TC-01 project file — all settings, bars, and sections.

Export scope: Current Playback or All Active Bars. All export formats require a Pro license.

Tips

  • - Use the metronome to verify rhythm alignment with syncopated or custom patterns.
  • - Preview rhythms in TR5 before applying them.
  • - Set different bar lengths in TR5 (e.g., 4/4 + 3/4) for polymetre.
  • - Ornaments work best on longer notes — very short notes may not have room.
  • - TR3 + TR4 interact: e.g., Arch + Stepwise = smooth arcs; Plateau + Leap = angular lines in a narrow range.
  • - Build Bars 1–2 in Free Mode, then switch to Phrase Mode to auto-generate Bars 3–8.
