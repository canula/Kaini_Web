<task type="auto">
  <name>Task 2: Create index.html with complete head metadata and Tailwind v4 CDN</name>
  <files>index.html</files>
  <action>
Create `index.html` with a complete `<head>` section and a minimal `<body>` placeholder. The body will be built out in Plan 02.

**DOCTYPE and html tag:**
- `<!DOCTYPE html>` + `<html lang="en">`

**Head section — in this exact order:**

1. **Charset and viewport:**
   ```html
   <meta charset="UTF-8" />
   <meta name="viewport" content="width=device-width, initial-scale=1.0" />
   ```

2. **Primary SEO (SEO-01):**
   - Title tag exactly: `Melody Generator App | TC-01 Tone Calculator`
   - Meta description: ease-focused messaging targeting beginners and hobbyist musicians. Include primary keyword "melody generator app" and secondary keywords "melody creator", "song maker app". Keep under 160 characters. Direction: "Generate melodies in seconds — no music theory needed. TC-01 Tone Calculator is a free melody generator app for Android, Windows, and Web."

3. **Canonical URL (SEO-04):**
   ```html
   <link rel="canonical" href="https://kaini-instruments.com/" />
   ```
   Must be absolute HTTPS URL.

4. **Open Graph tags (SEO-03):**
   ```html
   <meta property="og:type" content="website" />
   <meta property="og:title" content="Melody Generator App | TC-01 Tone Calculator" />
   <meta property="og:description" content="Generate melodies in seconds — no music theory needed." />
   <meta property="og:image" content="https://kaini-instruments.com/assets/images/og-image.png" />
   <meta property="og:url" content="https://kaini-instruments.com/" />
   <meta property="og:site_name" content="TC-01 Tone Calculator" />
   <meta property="og:locale" content="en_US" />
   ```
   Note: og:image MUST be absolute URL even though the image doesn't exist yet.

5. **Favicon and browser tab:**
   ```html
   <link rel="icon" href="/assets/images/favicon.png" type="image/png" />
   <link rel="apple-touch-icon" href="/assets/images/apple-touch-icon.png" />
   <meta name="theme-color" content="#0f172a" />
   ```
   Use a dark navy theme-color that matches the page background for immersive mobile feel.

6. **Google Fonts — Inter:**
   ```html
   <link rel="preconnect" href="https://fonts.googleapis.com" />
   <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
   <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet" />
   ```

7. **JSON-LD SoftwareApplication schema (SEO-05):**
   ```html
   <script type="application/ld+json">
   {
     "@context": "https://schema.org",
     "@type": "SoftwareApplication",
     "name": "TC-01 Tone Calculator",
     "applicationCategory": "MusicApplication",
     "operatingSystem": "Android, Windows, Web",
     "offers": {
       "@type": "Offer",
       "price": "0",
       "priceCurrency": "USD",
       "description": "Free download with optional Pro license key"
     },
     "url": "https://kaini-instruments.com/",
     "publisher": {
       "@type": "Organization",
       "name": "Kaini Instruments"
     }
   }
   </script>
   ```
   Place JSON-LD inside `<head>`, not `<body>`.

8. **Tailwind CSS v4 CDN (INFR-01):**
   ```html
   <script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>
   <style type="text/tailwindcss">
     @theme {
       --color-accent: #f97316;
       --color-accent-hover: #ea580c;
       --color-surface: #0f172a;
       --color-surface-alt: #1e293b;
       --color-heading: #f8fafc;
       --color-body: #94a3b8;
       --color-nav: #0f172a;
     }
   </style>
   ```
   - `--color-accent`: vibrant/electric orange (`#f97316` — Tailwind orange-500)
   - `--color-accent-hover`: slightly darker for hover states (`#ea580c` — orange-600)
   - `--color-surface`: deep navy/charcoal background (`#0f172a` — slate-900)
   - `--color-surface-alt`: slightly lighter for section alternation (`#1e293b` — slate-800)
   - `--color-heading`: near-white for headings (`#f8fafc` — slate-50)
   - `--color-body`: muted gray for body text (`#94a3b8` — slate-400)
   - `--color-nav`: nav background (same as surface for now)

**Body section — minimal placeholder only:**
```html
<body class="bg-surface text-body font-['Inter',sans-serif]">
  <!-- Nav, sections, and footer will be added in Plan 02 -->
</body>
```

Do NOT add any nav, sections, or footer content. Plan 02 handles the entire body.
